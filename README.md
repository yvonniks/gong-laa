# gong-laa
Cantonese vocabulary and phrases flashcards

Audio: Audio quality depends entirely on whether the user's device has a Cantonese (zh-HK / yue) voice installed. Falls back to Mandarin voice or shows "no Chinese voice found" if not — no bundled audio files.

Design look and feel: Hong Kong minibus destination-plate design - blue plate borders, red route-number accents, bolt corners, a mono/display type pairing.

My Cantonese language learning philosophy: production over recognition — actively generating the target language matters more than passive flashcard recognition. This shaped the EN→中 default direction and the phrase-first design. 

Work in progress. Please share any feedback or requests!
